testing exceptions
    we can use assertThrows to test if an exception has been thrown
    can check for specific exceptions

    example - trolley class
        @Test
        public void testAddNullItem() {
            Trolley t = new Trolley();
            Assertions.assertThrows(IllegalArgumentException.class, () -> {
                t.add(null);    // lambda expressions !!!
            });
        }
