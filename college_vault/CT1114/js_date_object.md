# The Date Object

**initialising**
```js
var <name> = new Date();
// calling the function to create a new date
```
## Methods
- .getFullYear()
gives year as 4-digit integer
- .getMonth()
gives month as 0-11
- .getDate()
gives day number as 1-31
- .getDay()
gives day number as 0-6
- .getHours()
gives hours as 0-23
- .getMinutes()
gives minutes as 0-59
- .getSeconds()
gives seconds as 0-59
